# ✏️ 文悦阁 - 精品文具商城

精品文具在线购买系统，纯前端 SPA，无需任何依赖。

## 功能特性

- ✏️ **13款精品文具** — 笔类/纸张类/收纳类/办公工具 4大品类
- 🔍 **分类筛选** — 4大品类一键过滤，实时关键词搜索（支持标签搜索）
- 🛒 **购物车侧边栏** — 增删改数量/实时总价/角标
- 📄 **商品详情弹窗** — 规格标注(如"1包500张"/"5本套装")/商品标签/库存
- 📦 **结算下单** — 表单验证（姓名/手机/地址）
- 💾 **库存管理** — 下单前验证库存，下单后自动扣减
- 🏷️ **促销标签** — 🔥热卖/✨新品/⚡特惠
- 📊 **排序功能** — 默认/价格升序/价格降序/销量优先
- 💾 **localStorage 持久化** — 购物车(stationery_cart)/库存(stationery_stock)/订单(stationery_orders)
- 📱 **响应式设计** — 桌面4列/平板2-3列/手机2列
- 📚 **清新文具主题** — 蓝白配色，学习氛围浓厚

## 技术栈

- HTML5 + CSS3 + Vanilla JS
- 纯前端 SPA，无后端依赖
- localStorage 数据持久化
